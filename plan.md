## 技术栈

- Vue 3.4 + TypeScript 5.0
- Vite 5.0 + pnpm
- Element Plus 2.5
- Pinia 2.1 + Vue Router 4.2
- Axios 1.6 + ECharts 5.x

### 后端
- Spring Boot 2.x + JDK 1.8
- TiDB 8.5 + MyBatis Plus
- JetCache + Lettuce (Redis)
- JWT 认证

### 开发工具
- ESLint + Prettier + Husky
- Vitest + Playwright
- Docker + Docker Compose

## 风险与挑战

### 技术风险
1. **TiDB 连接**: 通过 socks 代理连接可能不稳定
   - 缓解措施：配置连接池，实现自动重连
2. **Axure 原型实现**: 严格遵循原型可能增加开发难度
   - 缓解措施：与设计师沟通，适当调整不合理的设计
3. **性能优化**: 大量数据时可能出现性能问题
   - 缓解措施：提前进行性能测试，优化数据库查询

### 进度风险
1. **功能范围大**: 27 个 TODO 功能 + 后端开发
   - 缓解措施：分阶段交付，优先实现核心功能
2. **测试覆盖率要求高**: 单元测试 > 80%
   - 缓解措施：边开发边写测试，不要积压

### 资源风险
1. **开发人员**: 需要前后端全栈开发能力
   - 缓解措施：合理分配任务，必要时寻求支持
2. **测试环境**: 需要 TiDB 测试环境
   - 缓解措施：提前准备测试环境，确保连接稳定
